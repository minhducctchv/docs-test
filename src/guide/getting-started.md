# Getting started

## Prerequisites

- [Node.js 10+](https://nodejs.org/en/)
- [Yarn Classic](https://classic.yarnpkg.com/en/) (Optional)\*

cách viết ký tự đặc biệt \*

_cách viết chữ nghiêng_

## Quick start

<code-group>
<code-block title="dùng YARN" active>
```bash
yarn create vuepress-site [tên_project]
```
</code-block>

<code-block title="dùng NPM">
```bash
npx create-vuepress-site [tên_project]
```
</code-block>
</code-group>

Tiếp theo chương trình yêu cầu nhập: 

- Project Name
- Description
- Maintainer Email
- Maintainer Name
- Repository URL

## Run project

<code-group>
<code-block title="YARN" active>
```bash
cd docs
yarn install
yarn dev
```
</code-block>

<code-block title="NPM">
```bash
cd docs
npm install
npm run dev
```
</code-block>
</code-group>
